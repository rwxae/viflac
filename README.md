# viflac

Very simple FLAC metadata editor using [metaflac](https://github.com/xiph/flac) and `$EDITOR`.

> [!TIP]
> If you're looking for a general-purpose, feature-rich Audio metadata editor tool,
> I really recommend you to try [beets](https://beets.readthedocs.io/en/stable/index.html#).
> It is much more advanced program, that is able to automatically fetch songs metadata,
> fetch lyrics, organize library and many more. And if you still need to manually edit
> some tags, you can do that using [Edit Plugin](https://beets.readthedocs.io/en/stable/plugins/edit.html).
